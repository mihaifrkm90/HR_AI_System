# HR AI Evaluation System


## Descriere

HR AI Evaluation System este o aplicație web pentru evaluarea și monitorizarea candidaților folosind un sistem automat de scoring.

Aplicația permite departamentului HR să:

- adauge candidați;
- evalueze automat profilurile;
- urmărească evoluția performanței;
- adauge observații HR;
- genereze rapoarte.


## Tehnologii folosite

Backend:

- Python
- FastAPI


Frontend:

- HTML
- CSS
- JavaScript
- Chart.js


Stocare date:

- JSON Database



## Funcționalități


### Management candidați

- Adăugare candidat
- Editare candidat
- Ștergere candidat
- Căutare candidat
- Filtrare după nivel


### Evaluare AI

Sistemul calculează un scor pe baza:

- experienței;
- performanței;
- certificărilor.


Rezultatul include:

- scor;
- nivel candidat;
- recomandare HR.


### Monitorizare evoluție

Aplicația păstrează:

- istoricul evaluărilor;
- evoluția scorului;
- trendul candidatului.


### Dashboard

Include:

- număr total candidați;
- distribuție niveluri;
- scor mediu;
- grafice.


### Funcții HR

- comentarii interne;
- reevaluare manuală;
- raport candidat.
